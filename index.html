Mastermind Game

This project resembles a code-breaking game in which one guesses the correct series of random colored pegs. It provides feedback on the number of exact and inexact matches of each guess. The objective is to guess the correct code within the given rounds.

Flood-It! Game

This project is based off the strategy game where the goal is to flood the whole board with one color in less than the allowed steps. The user can customize and reset the board and steps.

Image Processing

This project is a program centered around layered image manipulation and enhancement. Using object-oriented design, it supports the ability to create images programmatically, import and export images, manipulate individual layers, add filter and color transformations, and display text-based and interactive GUI-based (Java Swing) user interfaces.

+

Freecell Card Game

This project is a text-based program that simulates a one-player card game. The objective is to sort all the given random batch of cards in 4 piles, where each pile must have the same suit and cards must be stacked in order from Ace to King.

+

Predator-Prey Simulation

This is a manual/automatic simulation coded in C++, that represents a predator (Doodlebug) chasing after their prey (Ant). The organisms move around with each time step and can breed once they reach maturity. They have different life spans, depending on how long they can last without starving or meeting their demise at their predators.

+

Messaging App

This is an android application (coded in Java using Android Studio) that enables users to create an account/login, take photos/retrieve from a gallery, chat with other existing users, and edit their profiles. Once an account is made, the user and chat information will be stored and authenticated in a cloud database (Firebase) that will be constantly updated.

Productivity App

This is an android application that encourages a logged-in user to study by letting them earn points after completing timed study sessions or lose points if their study sessions are interrupted. The program also keeps track of successful consecutive study sessions and the pet's mood which can lead to bonus points.

BGP Router

Developed router that accepts and forwards JSON data to desired destination. These messages tell the router how to forward data packets to far-flung destination on the internet.
